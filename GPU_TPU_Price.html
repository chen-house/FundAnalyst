阅读须知 · 价格口径与陷阱
  • 单位：NVIDIA 一律折算为 USD/GPU·小时（如 AWS p5.48xlarge 整机 8 卡 ÷ 8）；TPU 按 USD/chip·小时（一个 chip ≠ 一个 host，v5p 一个 host 含 4 chip）。
  • 按需 / 承诺 / Spot 三档差距巨大：以 H100 为例，AWS 按需 $12.29，1 年承诺约 $7.57，3 年承诺约 $4.93，Spot 可低至 $2-3 但随时可被回收。本页"按需"为列表价，不含企业谈判折扣（一般 30-50%）。
  • 整机 vs 单卡：Hyperscaler 通常只租 8 卡整机，单卡价是除算结果；Neocloud / 散户市场支持单卡按小时。前者实际起租成本远高于单卡价。
  • 区域差异：本页取北美区代表价。亚太 / 欧洲普遍贵 10-30%；中国区受出口管制有特殊型号（H20 / H200e），价格体系单独。
  • Spot 价格未纳入：Vast.ai / Salad 的中断价格波动剧烈，与不可中断价差距 50-80%，需独立追踪。
  • TPU 不能跨云租：TPU 仅 GCP 提供，无第三方分销；Trillium / Ironwood 部分区域需配额申请。
  • 静态快照局限：本页所有价格为人工录入快照，散户市场（RunPod / Vast）实际报价分钟级波动，建议使用前回到原站二次确认。
