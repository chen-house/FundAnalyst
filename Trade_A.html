月维度 · 行业景气方向

每月数据窗口发布后更新
宏观景气验证（每月固定时间查看，见下方日历）
NBS制造业PMI + 财新PMI
stats.gov.cn · 统计局最新发布
NBS官方PMI（大型国企为主）+ 财新PMI（中小民企为主），两者结合形成完整景气视图。重点看新订单指数和出厂价格指数分项。
月末+次月第1工作日官方首发
📐 >50扩张 · <50收缩
新订单-产成品库存差值为正 → 需求好于供给
财新显著高于NBS → 中小民企景气相对更好，利好中小盘
PPI / CPI 同比增速
stats.gov.cn · 统计局发布
PPI是工业企业盈利能力的先行指标，CPI反映消费端价格。PPI由负转正是上游资源/化工/钢铁盈利改善的直接信号。
每月约第9-10天官方
📐 PPI同比由负转正 → 上游周期股盈利拐点信号
PPI-CPI剪刀差收窄 → 中下游利润压力减小
CPI持续低迷 → 需求端偏弱，消费板块谨慎
M1 / M2 同比 · 社融规模
pbc.gov.cn · 货币金融统计
M1代表企业活期存款（经营意愿），M2代表广义货币。M1-M2剪刀差是判断企业资金活化程度的核心指标。
每月约第10-15天央行官方
📐 M1同比转正且加速 → 企业活期扩张，顺周期板块受益
社融超预期+居民中长贷改善 → 地产/消费链修复信号
社融持续低于预期 → 需求端疲弱，谨慎顺周期
规上工业企业利润 · 分行业
stats.gov.cn · 工业企业利润数据
统计局每月末发布规上工业企业利润，含分行业细项。是A股制造业上市公司盈利验证的直接数据，可判断哪些行业利润在加速改善。
每月约25-28日官方
📐 重点行业：计算机/电子、汽车、化工、钢铁、电力
利润增速连续3月改善 → 景气向上确认，可考虑加配
单月利润大幅转负 → 警惕盈利风险
申万行业指数 · 月度涨跌排名
swsresearch.com · 申万宏源行业指数
查看31个申万一级行业月度涨跌幅排名。结合基本面数据判断：资金认可的景气方向是否与经济数据一致，识别超涨或错杀板块。
月度汇总行业标准口径
📐 涨幅前5行业 → 当前资金景气偏好方向
基本面改善但涨幅靠后 → 可能存在错误定价机会
连涨3月+估值历史高位 → 警惕拥挤度风险
新能源汽车月度产销数据
caam.org.cn · 中汽协产销统计
新能源汽车月度销量及渗透率，是判断动力电池/正负极/铜箔等汽车产业链景气的中观指标，每月中旬发布上月数据。
每月约第8-10天行业协会官方
📐 渗透率突破新高 → 产业链量价齐升信号
销量环比连续下滑 → 库存积压，上游原材料谨慎
总销量同比增速加速 → 新能源全产业链受益

不定期官方
📐 重点追踪：集成电路产业发展规划、机器人/低空经济标准出台，是对应主题交易的催化节点
财联社电报 · 盘中实时快讯
cls.cn · 财联社实时资讯
A股影响力最大的实时快讯平台，"电报效应"对盘中个股情绪有直接影响。但准确率不稳定，重要信息须等官方原文确认。
实时情绪信号
⚠ 快讯只反映情绪，不作为独立决策依据。"消息人士称/某机构表示"类快讯需等官方文件确认后再操作
